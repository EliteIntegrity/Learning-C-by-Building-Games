# Vibe Asteroids — Chapter 25

The end-state of the vibe-coded Asteroids from Chapter 25 of *Learning
C++ by Building Games*. Single `main.cpp`, ~370 lines, classic
gameplay.

This is **my** finished version. Yours will look different — see the
chapter for the prompt-by-prompt journey through architecture,
implementation, and the messy middle.

## Controls

- **W / Up** — thrust
- **A / Left, D / Right** — rotate
- **Space** — fire
- **R** (after game over) — restart
- **Esc** — quit

Score and lives show in the title bar.

## To build

Empty C++ project per the Chapter 1 setup, then drop in `main.cpp`. Core
SDL 3 only.
