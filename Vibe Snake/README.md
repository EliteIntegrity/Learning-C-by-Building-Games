# Vibe Snake — Chapter 24

The end-state of the vibe-coded Snake from Chapter 24 of *Learning C++ by
Building Games*. Single `main.cpp`, classic gameplay.

This is **my** finished version. Yours will look different — the chapter
shows the prompt-by-prompt journey, but the AI's outputs are non-
deterministic and your tastes and pushback will differ from mine.

## Controls

- **WASD / arrows** — change direction
- **R** (after game over) — restart
- **Esc** — quit

Score shows in the title bar.

## To build

Empty C++ project per the Chapter 1 setup, then drop in `main.cpp`. Core
SDL 3 only.
